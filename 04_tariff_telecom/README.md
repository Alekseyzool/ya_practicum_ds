# Telecom Tariff

**Цель:** Сравнить прибыльность тарифов «Смарт» и «Ультра» и проверить влияние региона проживания абонентов.  
**Данные:** История использования 500 клиентов «Мегалайн» за 2018 год: звонки, SMS, трафик и тариф.  
**Подход:** Расчет выручки, описательная аналитика и статистические тесты (Welch t-test, z-тест) для тарифов и регионов.  
**Метрики/результаты:** Тариф «Ультра» приносит на ~60% больше выручки; различий между Москвой и регионами не выявлено (p≈0.18).

## Стек
`Python`, `Pandas`, `NumPy`, `Matplotlib`, `Seaborn`, `SciPy`

## Файлы
- `tariff_telecom.ipynb` — анализ тарифов и проверка гипотез по выручке.
