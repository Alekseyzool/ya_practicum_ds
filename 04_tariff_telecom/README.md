# Telecom Tariff

## Описание проекта
Аналитика клиентских данных оператора связи «Мегалайн»: сравнение тарифов «Смарт» и «Ультра» и оценка влияния региона на выручку.

## Цель
Выявить наиболее прибыльный тариф и подтвердить/опровергнуть гипотезы о различиях между сегментами клиентов.

## Стек технологий
`Python`, `Pandas`, `NumPy`, `Matplotlib`, `Seaborn`, `SciPy`

## Функциональность
- Расчёт ежемесячной выручки по каждому клиенту.
- Описательная аналитика и визуализация поведения абонентов.
- Статистические тесты (Welch t-test, z-тест) для сравнения тарифов и регионов.
- Формулировка рекомендаций по тарифной политике.

## Инструкция по запуску
1. Установите зависимости: `pip install -r requirements.txt`.
2. Подготовьте CSV-файлы с данными клиентов (ссылки указаны в ноутбуке) и разместите их рядом с `tariff_telecom.ipynb`.
3. Выполните ноутбук в Jupyter, следуя шагам очистки и анализа.
