# Oil Production Regions

## Описание проекта
Моделирование прибыльности трёх нефтяных регионов: прогнозируем объём запасов по скважинам и оцениваем риски при отборе лучших площадок.

## Цель
Выбрать регион, который приносит максимальную прибыль при ограничении риска убытка менее 2%.

## Стек технологий
`Python`, `Pandas`, `NumPy`, `scikit-learn`

## Функциональность
- Обучение моделей оценки запасов по признакам скважин.
- Расчет прибыли при выборе топ-200 скважин в каждом регионе.
- Bootstrap-анализ рисков и вероятность убытков.
- Формирование рекомендаций по инвестированию.

## Инструкция по запуску
1. Установите зависимости: `pip install -r requirements.txt`.
2. Поместите CSV `geo_data_0.csv`–`geo_data_2.csv` в каталог `/datasets/` (или обновите путь в ноутбуке).
3. Откройте `region_oil_production.ipynb` и выполните ячейки последовательно.
