# Oil Production Regions

**Цель:** Выбрать регион для бурения, обеспечивающий максимальную прибыль при допустимом риске.  
**Данные:** Геологоразведочные пробы трёх регионов (`geo_data_0.csv`–`geo_data_2.csv`) с признаками месторождений и объемом запасов.  
**Подход:** Обучение моделей для оценки запасов, вычисление прибыли по топ-200 скважин и bootstrap-анализ рисков.  
**Метрики/результаты:** Второй регион (`geo_data_1`) обеспечивает среднюю прибыль ≈10.5 млрд ₽ при риске убытка ~0.9% и рекомендован к разработке.

## Стек
`Python`, `Pandas`, `NumPy`, `scikit-learn`

## Файлы
- `region_oil_production.ipynb` — расчёт прибыли и оценка рисков по регионам.
