# Gold Refining

**Цель:** Спрогнозировать эффективность восстановления золота для оптимизации технологического процесса.  
**Данные:** Производственные логи обогащения с признаками потоков и целевыми показателями `rougher.output.recovery` и `final.output.recovery`.  
**Подход:** Очистка пропусков, проверка формулы восстановления, масштабирование признаков, обучение линейной регрессии, дерева и случайного леса с оценкой по sMAPE.  
**Метрики/результаты:** Лучшая модель (линейная регрессия) достигла sMAPE ≈ 8.29, что значительно лучше бейзлайна DummyRegressor.

## Стек
`Python`, `Pandas`, `NumPy`, `scikit-learn`

## Файлы
- `gold_refining.ipynb` — подготовка данных и построение модели прогноза восстановления.
