# Gold Refining

## Описание проекта
Моделирование эффективности технологической цепочки обогащения золота на основе производственных логов и расчёта sMAPE.

## Цель
Прогнозировать долю извлечения золота на промежуточном и финальном этапах, чтобы корректировать процесс и повысить прибыль.

## Стек технологий
`Python`, `Pandas`, `NumPy`, `scikit-learn`

## Функциональность
- Очистка данных, проверка корректности формулы восстановления.
- Масштабирование признаков и обучение нескольких моделей (Linear Regression, Decision Tree, Random Forest).
- Использование кастомной метрики sMAPE и сравнение с бейзлайном.
- Рекомендации по улучшению технологической цепочки.

## Инструкция по запуску
1. Установите зависимости: `pip install -r requirements.txt`.
2. Поместите входные CSV в каталог `/datasets/` (или обновите путь в ноутбуке).
3. Запустите `gold_refining.ipynb` и выполните ячейки для получения прогноза и аналитики.
