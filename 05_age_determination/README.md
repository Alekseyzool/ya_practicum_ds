# Age Determination

## Описание проекта
Камера в магазине фиксирует покупателей, а модель по фотографиям определяет возраст для выявления несовершеннолетних на витринах с алкоголем.

## Цель
Построить модель компьютерного зрения с точностью MAE < 8 лет на тестовой выборке.

## Стек технологий
`Python`, `TensorFlow`, `Pandas`, `NumPy`, `Matplotlib`, `Seaborn`

## Функциональность
- Подготовка датасета `/datasets/faces/` с аугментациями и генераторами изображений.
- Перенос обучения ResNet50, обучение верхних слоёв и fine-tuning.
- Мониторинг качества по MAE и визуализация результатов обучения.

## Инструкция по запуску
1. Установите зависимости: `pip install -r requirements.txt`.
2. Скопируйте датасет `/datasets/faces/` и файл `labels.csv` в рабочую среду.
3. Откройте `age_determination.ipynb`, выполните ячейки и сохраните модель при необходимости.
