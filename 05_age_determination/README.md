# Age Determination

**Цель:** Построить модель, определяющую возраст покупателей по фотографиям для контроля витрин.  
**Данные:** Размеченный датасет `/datasets/faces/` (7 591 изображение) с метками возраста в `labels.csv`.  
**Подход:** Генераторы изображений, перенос обучения ResNet50 и дообучение верхних слоёв на целевом датасете.  
**Метрики/результаты:** Получено MAE ≈ 6.32 года на тестовой выборке; рост ошибки после 9-й эпохи учтён при выборе лучшей модели.

## Стек
`Python`, `TensorFlow`, `Pandas`, `NumPy`, `Matplotlib`, `Seaborn`

## Файлы
- `age_determination.ipynb` — подготовка данных, обучение модели и оценка качества.

## Быстрый старт
- Установите зависимости: `pip install -r requirements.txt`.
- Загрузите датасет `/datasets/faces/` в рабочую среду и выполните ноутбук.
