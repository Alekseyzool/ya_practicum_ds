# Taxi Order Forecasting

## Описание проекта
Прогнозирование количества заказов такси на следующий час для аэропортов с использованием временных рядов и градиентного бустинга.

## Цель
Снизить простой автопарка и улучшить планирование графиков водителей, удерживая RMSE ниже целевого порога 48.

## Стек технологий
`Python`, `Pandas`, `NumPy`, `scikit-learn`, `CatBoost`, `Statsmodels`

## Функциональность
- Ресемплирование исходной минутной серии к часовому разрешению.
- Генерация лаговых признаков, скользящих средних и других статистик.
- Обучение моделей Linear Regression и CatBoost, сравнение с бейзлайном.
- Оценка качества по RMSE и выбор лучшего решения.

## Инструкция по запуску
1. Установите зависимости: `pip install -r requirements.txt`.
2. Поместите файл `taxi.csv` в каталог `/datasets/` или обновите путь в ноутбуке.
3. Запустите `taxi_order_forecasting.ipynb` и выполните ячейки в указанной последовательности.
