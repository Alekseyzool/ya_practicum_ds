# Taxi Order Forecasting

**Цель:** Прогнозировать число заказов такси на следующий час для аэропортов.  
**Данные:** Временной ряд `taxi.csv` с числом заказов `num_orders` и минутной/часовой детализацией.  
**Подход:** Ресемплирование до 1 часа, генерация лагов и скользящих средних, обучение линейной регрессии и CatBoost, сравнение с бейзлайнами.  
**Метрики/результаты:** Лучшая модель — линейная регрессия с RMSE ≈ 34 (ниже порога 48); CatBoost показал RMSE ≈ 37, а базовая стратегия — около 84.

## Стек
`Python`, `Pandas`, `NumPy`, `scikit-learn`, `CatBoost`, `Statsmodels`

## Файлы
- `taxi_order_forecasting.ipynb` — построение признаков и обучение моделей прогноза.

## Быстрый старт
- Установите зависимости: `pip install -r requirements.txt`.
- Поместите `taxi.csv` в `/datasets/` и пройдите ноутбук последовательно.
