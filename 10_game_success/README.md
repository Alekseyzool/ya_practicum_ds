# Game Success

## Описание проекта
Комплексный анализ рынка видеоигр: от динамики платформ до сравнения жанров и влияния пользовательских оценок на продажи.

## Цель
Найти закономерности продаж и подготовить рекомендации для маркетинговой кампании на 2017 год.

## Стек технологий
`Python`, `Pandas`, `NumPy`, `Matplotlib`, `Seaborn`, `SciPy`

## Функциональность
- Очистка и нормализация данных по продажам, платформам, оценкам и рейтингам ESRB.
- Исследование жизненных циклов платформ и распределения продаж по регионам.
- Сравнение жанров, корреляционный анализ оценок и проверка статистических гипотез.
- Формирование выводов и рекомендаций по приоритетным сегментам.

## Инструкция по запуску
1. Установите зависимости: `pip install -r requirements.txt`.
2. Загрузите датасет продаж (ссылка в ноутбуке) и разместите рядом с `game_success.ipynb`.
3. Выполните ноутбук в Jupyter, следуя шагам анализа и визуализаций.
