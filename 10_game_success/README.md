# Game Success

**Цель:** Найти закономерности успешных продаж видеоигр и подготовить план маркетинговой кампании на 2017 год.  
**Данные:** Исторические продажи игр с платформами, жанрами, региональными продажами, оценками пользователей и критиков, рейтингами ESRB.  
**Подход:** Очистка и нормализация данных, анализ жизненного цикла платформ, сравнение жанров по регионам, корреляция оценок и продаж, проверка гипотез.  
**Метрики/результаты:** Платформы PS4 и XOne остаются драйверами продаж; различий в пользовательских рейтингах Xbox One и PC не выявлено (p≈0.17), жанры Action и Sports статистически различаются.

## Стек
`Python`, `Pandas`, `NumPy`, `Matplotlib`, `Seaborn`, `SciPy`

## Файлы
- `game_success.ipynb` — аналитика рынка игр и проверка гипотез.

## Быстрый старт
- Установите зависимости: `pip install -r requirements.txt`.
- Подготовьте датасет продаж игр (см. ссылку в ноутбуке) и выполните анализ в Jupyter.
