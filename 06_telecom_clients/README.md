# Telecom Clients

**Цель:** Рекомендовать абоненту оптимальный тариф («Смарт» или «Ультра») по истории использования услуг.  
**Данные:** `users_behavior.csv` — агрегаты по звонкам, SMS и интернет-трафику с целевым признаком `is_ultra`.  
**Подход:** Разделение на обучающую, валидационную и тестовую выборки, обучение DecisionTree, RandomForest и LogisticRegression, подбор гиперпараметров по accuracy.  
**Метрики/результаты:** RandomForest показал accuracy ≈ 0.80; для продакшена предложено дерево решений глубиной 4 как быстрый и точный вариант.

## Стек
`Python`, `Pandas`, `scikit-learn`

## Файлы
- `telecom_clients.ipynb` — построение моделей и выбор лучшего тарифа для клиента.

## Быстрый старт
- Скачайте `users_behavior.csv` (см. ссылку в ноутбуке) и выполните анализ в Jupyter.
