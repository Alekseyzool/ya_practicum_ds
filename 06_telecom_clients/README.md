# Telecom Clients

## Описание проекта
Построение рекомендательной модели для оператора связи, которая по агрегированным признакам использования услуг выбирает подходящий тариф.

## Цель
Определить, какой тариф («Смарт» или «Ультра») предлагается клиенту, исходя из его поведения за прошлые месяцы.

## Стек технологий
`Python`, `Pandas`, `scikit-learn`

## Функциональность
- Разделение данных `users_behavior.csv` на обучающие/валидационные/тестовые выборки.
- Обучение моделей DecisionTree, RandomForest и LogisticRegression с подбором гиперпараметров.
- Сравнение accuracy и выбор компромисса между качеством и скоростью.
- Рекомендация модели для продакшена и интерпретация результатов.

## Инструкция по запуску
1. Установите зависимости: `pip install -r requirements.txt`.
2. Скачайте `users_behavior.csv` (ссылка в ноутбуке) и разместите рядом с `telecom_clients.ipynb`.
3. Запустите ноутбук в Jupyter и выполните ячейки последовательно.
