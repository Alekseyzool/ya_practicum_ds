# Car Cost

## Описание проекта
Модель оценки стоимости подержанных автомобилей по данным объявлений: технические характеристики, комплектация, возраст и пробег.

## Цель
Построить прогноз цен, чтобы ускорить подготовку предложений и снизить риск неправильной оценки.

## Стек технологий
`Python`, `Pandas`, `NumPy`, `scikit-learn`, `LightGBM`, `CatBoost`

## Функциональность
- Очистка и подготовка исторических объявлений, обработка пропусков и категориальных признаков.
- Обучение и сравнение моделей Linear Regression, LightGBM и CatBoost по RMSE и скорости.
- Анализ важности признаков и рекомендации по внедрению.

## Инструкция по запуску
1. Установите зависимости: `pip install -r requirements.txt`.
2. Загрузите исходный датасет (ссылка в ноутбуке) и поместите рядом с `car_cost.ipynb`.
3. Откройте ноутбук в Jupyter и выполните ячейки для подготовки данных и обучения моделей.
