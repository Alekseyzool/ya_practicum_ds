# Car Cost

**Цель:** Построить модель оценки стоимости автомобилей для сервиса «Не бит, не крашен».  
**Данные:** Исторические объявления с техническими характеристиками, датами и ценами.  
**Подход:** Очистка данных, кодирование признаков, обучение и сравнение LinearRegression, LightGBM и CatBoost с учётом времени обучения и предсказания.  
**Метрики/результаты:** LightGBM обеспечивает RMSE ≈ 1613 €, балансируя качество и скорость; CatBoost даёт минимальную ошибку (≈1587 €), но дольше обучается.

## Стек
`Python`, `Pandas`, `NumPy`, `scikit-learn`, `LightGBM`, `CatBoost`

## Файлы
- `car_cost.ipynb` — подготовка данных и сравнение моделей оценки цены.

## Быстрый старт
- Установите зависимости: `pip install -r requirements.txt`.
- Загрузите исходный датасет (см. ноутбук) и выполните ячейки анализа.
