# Bank Borrowers

## Описание проекта
Исследование платёжной дисциплины клиентов банка: анализ влияния семейного статуса, наличия детей, уровня дохода и цели кредита на вероятность просрочки.

## Цель
Определить факторы, повышающие риск невозврата и сформировать рекомендации по скоринговой политике.

## Стек технологий
`Python`, `Pandas`

## Функциональность
- Очистка пропусков, обработка аномальных значений и категоризация признаков.
- Построение сводных таблиц и проверка статистических гипотез.
- Вывод ключевых факторов риска и формулировка рекомендаций.

## Инструкция по запуску
1. Установите зависимости: `pip install -r requirements.txt`.
2. Разместите исходный датасет в директории рядом с ноутбуком (см. путь в первых ячейках).
3. Откройте `bank_borrowers.ipynb` и выполните анализ шаг за шагом.
