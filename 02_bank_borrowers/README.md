# Bank Borrowers

**Цель:** Оценить влияние семейного положения, детей, дохода и цели кредита на возврат займа в срок.  
**Данные:** Клиентская таблица банка с признаками `children`, `family_status`, `income_type`, `total_income`, `purpose` и флагом просрочки.  
**Подход:** Очистка пропусков и аномалий, категоризация доходов и целей, построение сводных таблиц и проверка статистических гипотез.  
**Метрики/результаты:** Подтверждено влияние детей, семейного статуса и цели кредита на просрочки; уровень дохода статистически значимо не влияет на возврат.

## Стек
`Python`, `Pandas`

## Файлы
- `bank_borrowers.ipynb` — подготовка данных и проверка гипотез по просрочкам.

## Быстрый старт
- Установите зависимости: `pip install -r requirements.txt`.
- Откройте ноутбук в Jupyter и выполните анализ шаг за шагом.
