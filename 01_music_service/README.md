# Music Service

## Описание проекта
Анализ прослушиваний Яндекс.Музыки в двух городах: сравниваем активность и жанровые предпочтения в Москве и Санкт-Петербурге.

## Цель
Проверить гипотезы о различиях в поведении пользователей в зависимости от города, времени суток и дня недели.

## Стек технологий
`Python`, `Pandas`

## Функциональность
- Предобработка и очистка журнала прослушиваний `yandex_music_project.csv`.
- Разрез по городам и дням недели, сравнение активности утром и вечером.
- Проверка жанровых различий и формулировка выводов по гипотезам.

## Инструкция по запуску
1. Установите зависимости: `pip install -r requirements.txt`.
2. Скопируйте датасет `yandex_music_project.csv` в директорию проекта.
3. Откройте ноутбук `music_service.ipynb` в Jupyter и выполните ячейки последовательно.
