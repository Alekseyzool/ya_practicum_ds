# Music Service

**Цель:** Проверить гипотезы о различиях в поведении пользователей Яндекс.Музыки в Москве и Санкт-Петербурге.  
**Данные:** Логи прослушиваний `yandex_music_project.csv` с отметками города, времени и жанра.  
**Подход:** Предобработка данных, группировки по городам и дням недели, сравнение жанров и активности пользователей.  
**Метрики/результаты:** Подтверждено влияние дня недели на активность по городам; жанровые различия минимальны, вторая гипотеза подтверждена частично.

## Стек
`Python`, `Pandas`

## Файлы
- `music_service.ipynb` — полный анализ поведения слушателей и проверка гипотез.
