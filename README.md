# Проекты Яндекс Практикума

## Описание проекта
Коллекция учебных работ из программы «Специалист по Data Science» Яндекс Практикума: от базового анализа данных до моделей машинного обучения, временных рядов и обработки текста.

## Цель
Систематизировать выполненные проекты и предоставить быстрый доступ к ноутбукам, данным и стеку для воспроизведения результатов.

## Стек технологий
`Python`, `Pandas`, `NumPy`, `Matplotlib`, `Seaborn`, `scikit-learn`, `CatBoost`, `LightGBM`, `TensorFlow`, `Statsmodels`, `NLTK`

## Инструкция по запуску
1. Перейдите в папку интересующего проекта из таблицы.
2. Установите зависимости: `pip install -r requirements.txt`.
3. Скачайте требуемые датасеты (ссылки указаны в README проекта) и разместите их в указанном расположении.
4. Откройте ноутбук в Jupyter/VS Code и выполните ячейки последовательно.
